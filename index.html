Fossil Fuel Application



  • Fossil Fuel Application

    R Shiny Application.

Web Application built in R with Shiny. This app analyzes global Fossil Fuel Emissions, including global trends and time series forecasts, country specific trends, and industry specific trends. The data is from Our World in Data's github repository. The application is hosted on the shinyapps.io cloud.

COVID-19 Dashboard



  • COVID-19 Dashboard

    Python Dash Application.

Python Dash Flask App that visualizes daily COVID-19 data. This Dashboard analyzes global spread and vacination trends, utilizes ARIMA modeling to forecast global vaccination rates, and uses cumulative, daily, per capita, and smoothed time series datasets to compare the impact of the virus by country. The data for COVID-19 spread and vaccinations are from CSSE and Centers for Civic Impact at Johns Hopkins University respectively. The Dashboard has been deployed using Heroku.

NBA Web Scraper



  • NBA Web Scraper

    Python Dash Application.

A Python Web Scraper built using Dash. It scrapes per game NBA data for each player from basketball-reference by using BeautifulSoup. NBA data can be scraped from 1980-present, and is sorted alphabetically. The data table is interactive and filterable, and is linked to three column charts. The Dash App is hosted on Heroku.

Vaccine Sentiments



  • Vaccine Sentiments

    R Markdown notebook.

R notebook that pulls tweets using the Twitter Api and performs sentiment analysis. The notebook displays unique word counts and sentiment analysis charts using the Bing and NRC lexicons for the hashtags #covid19 and #vaccine. The purpose is to understand and visualize the sentiments from Twitter users regarding COVID-19 and the vaccines.

Census Analysis



  • Census Analysis

    Python Jupyter notebook.

Jupyter notebook written in Python that analyzes certain variables from the Census's ACS 5-year estimates. The notebook pulls 2018 data from the Census Api and analyzes nationwide unemployment, inequality, and income. For each indicator, it visualizes the country and state distributions and the relationships between these variables. The notebook performs K-means clustering to split counties into income quartiles and analyze these relationships by cluster.
